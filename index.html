📲 Cara Install Aplikasi

Kamu bisa install langsung dari sini:

— atau ikuti langkah manual —
1

2
Ketuk ikon ⋮ (tiga titik) di pojok kanan atas Chrome
3
Pilih "Tambahkan ke Layar Utama" atau "Install App"
4
Ketuk "Tambah" / "Install" pada dialog yang muncul
5
Aplikasi muncul di layar utama seperti app biasa ✅
💡 Tips: Jika menu "Tambahkan ke Layar Utama" tidak muncul, pastikan file dibuka via HTTP/HTTPS bukan langsung dari file manager.

4
Ubah nama jika perlu, lalu ketuk "Add"
5
Ikon aplikasi muncul di Home Screen ✅
